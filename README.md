# 📅 מנהל משמרות (Shift Manager)

אפליקציית ניהול משמרות עבודה בנויה עם React, Vite, ו-Tailwind CSS. האפליקציה תומכת בעברית עם כיוון RTL וחוסכת את כל הנתונים ב-localStorage.

## ✨ תכונות

- ✅ ממשק בעברית עם כיוון RTL
- 📱 רספונסיבי (מובייל, טבלט, דסקטופ)
- 💾 שמירת נתונים ב-localStorage (ללא צורך ב-backend)
- 🎨 עיצוב מודרני עם Tailwind CSS
- ⚡ ביצועים מהירים עם Vite

## 🗂️ מבנה הפרויקט

```
src/
├── components/    # קומפוננטות React
├── hooks/        # Hook מותאמים (useLocalStorage)
├── utils/        # פונקציות עזר
├── data/         # קבצי נתונים (employees.js)
├── App.tsx       # קומפוננט ראשי
└── index.css     # סגנונות גלובליים
```

## 🚀 התחלה מהירה

### דרישות מקדימות
- Node.js 16+ ו-npm 7+

### התקנה

1. **קלון/פתיחה של הפרויקט:**
   ```bash
   cd shift-manager
   ```

2. **התקן את התלויות:**
   ```bash
   npm install
   ```

3. **הרץ את שרת הפיתוח:**
   ```bash
   npm run dev
   ```

   האפליקציה תיפתח בדפדפן ב-`http://localhost:5173` (או במעבר אחר זמין)

## 📦 פקודות זמינות

```bash
# הרץ את שרת הפיתוח
npm run dev

# בנה לייצור
npm run build

# תצוגה מקומית של הבנייה
npm run preview

# בדיקת eslint
npm run lint
```

## 📋 נתונים לדוגמה

הקובץ `src/data/employees.js` מכיל מערך של עובדות לדוגמה עם השדות הבאים:

```javascript
{
  id: 1,                    // מזהה ייחודי
  name: "שם העובדת",        // שם מלא
  weeklyShift: 40,          // שעות שבועיות
  friday: true,             // עבודה בשישי
  shiftType: "הכל",         // סוג משמרת: "הכל", "בוקר", "ערב", "אמצע"
  availableFrom: "08:00",   // זמן התחלה
  availableTo: "20:00"      // זמן סיום
}
```

## 💾 שמירת נתונים

כל שינוי בנתונים נשמר באופן אוטומטי ב-localStorage של הדפדפן. הנתונים יישמרו גם לאחר סגירה ופתיחה מחדש של הדפדפן.

## 🎨 התאמה

### שינוי צבעים
ערוך את קובץ `tailwind.config.js` לשינוי הצבעים והעיצוב.

### הוספת עובדות
עדכן את `src/data/employees.js` או שנה הנתונים דרך הממשק.

### הוספת תכונות
בנה קומפוננטות חדשות בתיקייה `src/components/`.

## 🔗 טכנולוגיות

- **React 18** - ספרייה לבניית UI
- **TypeScript** - בטיחות סוגים
- **Vite** - כלי בנייה מהיר
- **Tailwind CSS** - עיצוב יחסי
- **localStorage** - שמירת נתונים לוקלית

## 📝 הערות

- אין צורך במחשב (backend) - האפליקציה עובדת בלי תקשורת חיצונית
- כל הנתונים מאוחסנים בלבד במכשיר המשתמש
- טח לשימוש בודד או בצוות קטן

## 🛠️ פתרון בעיות

אם אתה מתמודד עם בעיות:

1. **נקה את npm cache:**
   ```bash
   npm cache clean --force
   npm install
   ```

2. **מחק את node_modules וה-lockfile:**
   ```bash
   rm -r node_modules package-lock.json
   npm install
   ```

3. **בדוק גרסה של Node.js:**
   ```bash
   node --version  # צריך להיות 16 ומעלה
   ```

למידע נוסף, ראה את התיקיות `src/` ותעיין בקבצים ישירות.
